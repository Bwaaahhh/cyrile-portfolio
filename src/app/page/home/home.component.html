Bienvenue sur mon portfolio

Fort d'une expérience dans différents domaines je me suis tourné vers le développement web qui réunissait nombres de points que je recherche dans un travail.

J'ai pu grâce à l'Access Code School apprendre les bases nécessaires puis j'ai consolidé mes connaissances en faisant un stage de 8 semaines à l'agence SpinOn à Besançon.

Cette application a été réalisée avec le framework Angular et utilise Symfony pour la gestion de la base de donnée. De plus, j'ai utilisé du Scss, Foundation et GruntJS pour une facilité de production et de maintenabilité.

Si vous souhaitez collaborer avec moi pour un futur projet, n'hésitez pas à me contacter :)
